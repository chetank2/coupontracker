# MLC-LLM Native Artifacts

Place the headers and shared libraries generated by the MLC toolchain in this directory so the Android build can consume them deterministically.

```
app/libs/mlc_llm/
├── include/                  # Top-level headers such as mlc/runtime/c_runtime_api.h
├── 3rdparty/
│   ├── dlpack/include/       # DLPack headers
│   └── tvm/include/          # TVM runtime headers
└── lib/
    ├── arm64-v8a/            # libmlc_llm_runtime.so, libtvm_runtime.so, librelaxt_runtime.so, etc.
    ├── armeabi-v7a/
    └── x86_64/
```

The Gradle build expects these folders to contain the real binaries for each ABI. Use the `BUILD_MOCK_JNI` CMake option if you need to fall back to the mock implementation.
