Privacy Policy

Effective date: 2026-05-06

Coupon Tracker is designed to help users save and manage coupons from screenshots and images. This policy explains what data the app accesses, how it is used, and what control users have over that data.

Information Stored in the App

  • Coupon details such as store name, offer description, coupon code, redeem code, and expiry date
  • App preferences such as theme, reminders, onboarding state, and model setup state
  • Optional local diagnostics or logs that remain on the device unless the user exports them

Permissions and Access

  • Camera access when the user chooses to scan a coupon
  • Photos or file access when the user imports an image or restores a backup
  • Notification access for expiry reminders
  • Network access only when the user chooses to download model files or related assets

How Data Is Used

  • To extract coupon information from screenshots or images provided by the user
  • To store coupons locally and show reminders before expiry
  • To download model files explicitly requested by the user

Transparency and Limitations

  • Coupon Tracker is a personal organization and reminder tool. It is not a financial service, payment service, gambling app, lending app, or investment app.
  • Coupon extraction uses OCR, rules, and optional on-device models. Results can be incomplete or wrong, so users should review coupon details before saving or redeeming them.
  • Coupon Tracker does not guarantee that a coupon is valid, redeemable, accepted by a merchant, or that it will produce any savings.
  • Coupon Tracker is not affiliated with the stores, brands, or coupon issuers that may appear in screenshots unless explicitly stated elsewhere.

Data Sharing

Coupon Tracker is intended to process coupon data on device after model setup. Coupon screenshots and coupon content are not intended to be uploaded to the developer’s servers as part of normal offline extraction.

If a user exports a backup or shares app content, that action is controlled by the user and sent only to the destination they choose.

Security

  • Model downloads are intended to use HTTPS
  • Locally stored data remains on the device unless the user exports or shares it

Retention and Deletion

Data remains on the device until the user deletes coupons, deletes model files, clears app data, restores over existing data, or uninstalls the app.

Children

Coupon Tracker is not intended for children and does not knowingly collect personal data from children.
