# Mobile Coupon Trainer PWA

A complete Progressive Web App for offline coupon annotation and training on mobile devices.

## Features
- 📱 Mobile-first PWA with offline functionality
- ✍️ Touch-based annotation system
- 💾 Local storage with IndexedDB
- 🎯 Service Worker for offline capability
- 📤 Upload interface with drag & drop
- 📊 Data management and export
- 🏠 Install as native app

## Usage
1. Serve the files locally: `python3 -m http.server 8080`
2. Open http://localhost:8080 in your browser
3. Install as PWA on mobile devices
4. Use offline for coupon annotation and training

Built for the CouponTracker project.
