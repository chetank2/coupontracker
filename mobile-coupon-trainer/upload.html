Stage 1: Coupon Boundaries

  4. Switch to Stage 2 when all coupons are marked

Stage 2: Field Detection

  1. Select a coupon instance to annotate
  2. Draw rectangles around specific fields (code, benefit, expiry, etc.)
  3. Repeat for each coupon instance
  4. Save when all fields are annotated
